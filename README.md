#Project Name: Expense Tracker
#Objective: 
    To create a program that records expenses and calculates the total amount spent.
#Skills Used:
  Variables (total)
  User Input (input())
  Loops (while)
  Conditional Statements (if)
  Arithmetic Operations (+)
#How It Works:
  Start with a total of 0.
  Ask the user to enter an expense.
  Add the expense to the total.
  Repeat until the user enters 0.
  Display the final total spent.
